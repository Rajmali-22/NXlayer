================================================================================
                    AI TEXT BOT - SETUP GUIDE FOR NEW USER
================================================================================

STEP 1: COPY PROJECT FOLDER
----------------------------
Copy the entire "ai-text-bot" folder to your computer.
Example: C:\Users\YourName\Desktop\ai-text-bot


STEP 2: INSTALL PREREQUISITES
-----------------------------

A) Install Node.js (v16 or higher)
   - Download from: https://nodejs.org/
   - Choose "LTS" version (recommended)
   - Run installer, click Next through all steps
   - Restart your computer after installation

B) Install Python (v3.8 or higher)
   - Download from: https://www.python.org/downloads/
   - IMPORTANT: Check "Add Python to PATH" during installation
   - Click "Install Now"
   - Restart your computer after installation


STEP 3: VERIFY INSTALLATIONS
----------------------------
Open Command Prompt (cmd) and run:

   node --version
   (Should show v16.x.x or higher)

   python --version
   (Should show Python 3.8.x or higher)


STEP 4: INSTALL PROJECT DEPENDENCIES
-------------------------------------
Open Command Prompt and navigate to the project folder:

   cd C:\Users\YourName\Desktop\ai-text-bot

Install Node.js dependencies:

   npm install

Install Python dependencies:

   pip install -r requirements.txt


STEP 5: SET UP API KEYS
-----------------------
Open the ".env" file in the project folder with Notepad.

You need these API keys:

A) MISTRAL_API_KEY (Required - for text AI)
   - Go to: https://console.mistral.ai/api-keys/
   - Create account and generate API key
   - Paste it after: MISTRAL_API_KEY=

B) ANTHROPIC_API_KEY (Required - for screenshot vision)
   - Go to: https://console.anthropic.com/settings/keys
   - Create account and generate API key
   - Paste it after: ANTHROPIC_API_KEY=

C) OPENAI_API_KEY (Optional - for voice-to-text)
   - Go to: https://platform.openai.com/api-keys
   - Create account and generate API key
   - Paste it after: OPENAI_API_KEY=

Save the .env file after adding your keys.


STEP 6: RUN THE APPLICATION
---------------------------
Open Command Prompt in the project folder:

   cd C:\Users\YourName\Desktop\ai-text-bot
   npm start

The app will start and run in the background.


STEP 7: HOW TO USE
------------------

KEYBOARD SHORTCUTS:

1. Ctrl+Alt+Enter - Fix grammar/spelling of text you just typed
   - Type some text in any app
   - Press Ctrl+Alt+Enter
   - AI fixes your text and types it back

2. Ctrl+Alt+Enter (Double press) - Extend/continue your writing
   - After first Ctrl+Alt+Enter, press again within 2 seconds
   - AI continues writing from where you left off

3. Ctrl+Shift+D - Process clipboard content
   - Copy any text (Ctrl+C)
   - Press Ctrl+Shift+D
   - AI processes it (fixes code, answers questions, etc.)

4. Ctrl+Shift+F - Screenshot + Vision
   - Press Ctrl+Shift+F
   - Type what you want to know about the screen
   - Press Enter
   - AI analyzes your screen and responds

5. Ctrl+Shift+P - Paste/inject AI response
   - After AI generates response, press this to type it out

6. Escape - Cancel/close any popup

7. Ctrl+. (Ctrl + Period) - Pause/resume text injection


HUMANIZE TYPING:
- Toggle the "Human" switch in the overlay window
- When ON: Types with variable speed, occasional typos (looks human)
- When OFF: Fast direct typing (default)


TROUBLESHOOTING:
----------------

Problem: "npm not found"
Solution: Reinstall Node.js and restart computer

Problem: "python not found"
Solution: Reinstall Python, make sure to check "Add to PATH"

Problem: "Module not found" error
Solution: Run "pip install -r requirements.txt" again

Problem: Shortcuts not working
Solution: Run the app as Administrator

Problem: API errors
Solution: Check your API keys in .env file are correct
